Calls the doThis function the specified number of times.

based on ruby's 3.times {}

If step is not provided, it defaults to 1.

Implementation

void times({required void Function(int i) doThis, int? step}) {
  step ??= 1;
  for (var i = 0; i < this; i += step) {
    doThis(i);
  }
